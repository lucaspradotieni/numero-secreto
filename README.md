# numero-secreto

Objetivo:
- Um jogo onde o usuário tem que adivinhar o número gerado.

Como Funciona:
- O Sistema gera um número aleatório entre 1 e 10.
- O jogador chuta um número.
- O jogo informa se o número é maior, menor ou está correto.
- Quando se acerta o número, é mostrado a quantidade de tentativas.

O que foi usado:
- JavaScript
- HTML
- CSS

Como Iniciar:
1. Baixe a pasta de arquivos.
2. Execute o arquivo index.html no navegador.
